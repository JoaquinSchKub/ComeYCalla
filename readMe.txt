# 🍽️ Come y Calla

**Come y Calla** es un recetario web pensado para personas que se están independizando, conviviendo por primera vez o simplemente cansadas de comer siempre lo mismo. El sitio te ayuda a encontrar recetas originales, fáciles de preparar y con un toque de humor, para que cocinar no sea un castigo.

## 💡 ¿Por qué este proyecto?

Porque todos pasamos (o vamos a pasar) por esa etapa de mirar la heladera y pensar: *“otra vez delivery...”*. Este proyecto nace para acompañar esos momentos y ofrecer una experiencia más amigable, visual y divertida al elegir qué cocinar cada semana.

## 🧾 Características principales

- ✅ Página principal con recetas cargadas dinámicamente desde una API externa.
- ✅ Posibilidad de guardar recetas favoritas en tu propio recetario.
- ✅ Visualización de recetas guardadas con ingredientes y pasos.
- ✅ Funcionalidad para eliminar recetas o "comprar" el recetario.
- ✅ Estética clara y cálida, con colores pastel y diseño responsive.
- ✅ Formulario de contacto y reseña para que los usuarios compartan su experiencia.

## 📁 Estructura del proyecto

📦 Come-y-Calla/
├── index.html # Página principal
├── recetario.html # Página del recetario (recetas guardadas)
├── style.css # Estilos personalizados en tonos pastel
├── app.js # Lógica para cargar y guardar recetas
├── recetario.js # Lógica para mostrar y gestionar el recetario
├── img/ # Íconos e imágenes usadas


## 🧪 ¿Cómo usarlo?

1. Cloná el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/come-y-calla.git

2. Abrí el archivo index.html en tu navegador.

3. Navegá, elegí recetas y creá tu recetario.

4. Dejanos una reseña desde la sección de contacto ❤️

📦 Tecnologías usadas

	HTML5 + CSS3

	JavaScript Vanilla

	DummyJSON API para recetas de ejemplo

	localStorage para persistencia local
